# Spectrum — Generative Emotion Archive

Каждый день мы чувствуем десятки эмоций, но редко можем их выразить словами. Spectrum позволяет превратить чувства в визуальную форму — интерактивный 3D-куб, который меняет цвет, форму, прозрачность и движение в зависимости от того, что вы переживаете. Сохраните свою эмоцию в архив и исследуйте, что чувствуют другие люди.

## Возможности

- **Создание эмоций** — настройте 5 параметров (оттенок, прозрачность, вращение, шум, частицы) и наблюдайте как куб реагирует в реальном времени
- **Просмотр архива** — загружайте случайные эмоции других пользователей с амбиентным звуковым сопровождением
- **Регистрация и авторство** — зарегистрируйтесь, чтобы ваши эмоции подписывались вашим никнеймом
- **Автоматическое определение эмоций** — система распознает 15 эмоций (радость, меланхолия, ярость, спокойствие и др.) по комбинации параметров
- **Двуязычный интерфейс** — русский и английский

## Технологии

- **Frontend**: TypeScript, Three.js, Vite
- **Backend**: Python, FastAPI, asyncpg
- **БД**: PostgreSQL
- **Авторизация**: JWT + bcrypt

## Установка

### Требования

- Node.js 18+
- Python 3.10+
- PostgreSQL

### Backend

```bash
cd backend
python -m venv venv

# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

pip install -r requirements.txt

# Настройте подключение к БД
cp .env.example .env
# Отредактируйте .env — укажите строку подключения к PostgreSQL

# Запуск сервера (таблицы создаются автоматически)
uvicorn main:app --reload
```

API доступен на `http://localhost:8000`.

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.

## Использование

1. **Регистрация** (`#/auth`) — создайте аккаунт (никнейм + пароль)
2. **Вход** — авторизуйтесь, чтобы сохранять эмоции
3. **Создание** (`#/create`) — двигайте слайдеры, нажмите "Сохранить эмоцию"
4. **Просмотр** (`#/view`) — листайте случайные эмоции из архива, видите автора и определённую эмоцию

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/register` | Регистрация (username, password) |
| POST | `/login` | Вход — возвращает JWT-токен |
| POST | `/emotions` | Сохранить эмоцию (требует токен) |
| GET | `/emotions/random` | Получить случайную эмоцию |
